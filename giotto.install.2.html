Installation

Giotto consists of two modules: analyzer and viewer. The requirements of Giotto installation are:
  • R (at least 3.5.1)
  • Python (at least 3.0)
  • Java 1.6+
  • Unix / Linux environment
There are two options to install Giotto: 1) native install, or 2) docker image. Docker image is recommended for fast deployment, and is compatible across all operating systems. Native install may require a bit of effort (especially for macOS).

Compatibility

OS Native install Docker image
Ubuntu Linux Yes Yes
macOS X Yes Yes
Windows 10 Partial* Yes
* - Analyzer might work. Viewer requires linux tools.
Docker image is recommended for Windows.
