print.giotto {Giotto}R Documentation

print method for giotto class

Description

print method for giotto class. Prints the chosen number of genes (rows) and cells (columns) from the raw count matrix. Also print the spatial locations for the chosen number of cells.

Usage

print.giotto(object, ...)

Arguments

nr_genes

number of genes (rows) to print

nr_cells

number of cells (columns) to print
