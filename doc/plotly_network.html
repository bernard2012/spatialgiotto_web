plotly_network

Description

provide network segment to draw in 3D plot_ly()

Usage

plotly_network(
  network,
  x = "sdimx_begin",
  y = "sdimy_begin",
  z = "sdimz_begin",
  x_end = "sdimx_end",
  y_end = "sdimy_end",
  z_end = "sdimz_end"
)

Arguments

gobject

network in giotto object

Value

edges in network as data.table()

Examples

    plotly_network(gobject)
