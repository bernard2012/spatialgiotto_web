create_delaunayNetwork3D

Description

Create a spatial Delaunay network.

Usage

create_delaunayNetwork3D(
  gobject,
  method = "delaunayn_geometry",
  sdimx = "sdimx",
  sdimy = "sdimy",
  sdimz = "sdimz",
  name = "delaunay_network_3D",
  maximum_distance = "auto",
  minimum_k = 0,
  options = "Pp",
  return_gobject = TRUE,
  ...
)

Examples

    create_delaunayNetwork3D(gobject)
