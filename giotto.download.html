Download

Giotto consists of two modules: analyzer and viewer.

The analyzer is an R package named Giotto, available under https://github.com/RubD/Giotto. The installation page describes how to get the latest version of Giotto. We recommend getting the R Giotto package directly from within R in a remote install: "remotes::install_github("RubD/Giotto")". In addition to the R package, you also need to install some python packages (see Installation for details).

The viewer is a set of two Python packages (in pip repository): smfish-image-processing, and giotto-viewer. It requires opencv-python, scikit-image as pre-requisites. Also requires ImageMagick commonly available in Linux, and Ruby, tileup Ruby package. See Installation for details.

Archival: Giotto analyzer as tar-ball: Giotto-master.zip (450MB).

Docker image

As an alternative to the above downloads, to further simplify the install process, we also provide a premade docker image containing both the latest version of the analyzer and the viewer.
